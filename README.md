# simple-qdrant-project

Проект-визитка для экспериментов с векторной базой данных Qdrant.

## Описание

Этот проект демонстрирует работу с векторной базой данных [Qdrant](https://qdrant.tech/) для хранения и поиска по векторным представлениям текста. Проект индексирует FAQ-вопросы, преобразуя их в векторы с помощью Sentence Transformers, и сохраняет в Qdrant для последующего семантического поиска.

## Функциональность

- Загрузка предобученной модели `all-MiniLM-L6-v2` из библиотеки `sentence-transformers`
- Подключение к локальному экземпляру Qdrant (порт 6333)
- Создание коллекции `faq_collection` с векторами размерности 384 и косинусной мерой близости
- Чтение и обработка текстовых данных из `data/faq.txt`
- Векторизация каждого абзаца и загрузка в Qdrant с сохранением исходного текста в `payload`
- Подсчёт и вывод количества проиндексированных абзацев
- Поиск по запросу с возвратом наиболее релевантных ответов и их оценок

## Структура проекта

```
simple-qdrant-project/
├── README.md
├── docker-compose-qdrant-only.yml    # Конфигурация для запуска Qdrant через Docker
├── src/
│   ├── data/
│   │   └── faq.txt                   # Файл с FAQ-вопросами
│   ├── index_faq.py                  # Скрипт для индексации FAQ
│   └── askdoc.py                     # Скрипт для поиска ответов по векторному сходству
```

## Запуск проекта

1. Убедитесь, что Docker установлен и запущен.
2. Запустите Qdrant с помощью команды:
   ```bash
   docker-compose -f docker-compose-qdrant-only.yml up -d
   ```
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
   (Если файл `requirements.txt` отсутствует, установите вручную: `pip install qdrant-client sentence-transformers`)
4. Запустите скрипт индексации:
   ```bash
   python src/index_faq.py
   ```
5. Запустите интерактивный поиск:
   ```bash
   python src/askdoc.py
   ```

## Использование askdoc.py

После запуска `askdoc.py`, введите вопрос в командной строке. Программа вернёт до 3 наиболее релевантных ответов с указанием степени сходства (релевантности). Для выхода введите `exit`, `quit` или `выйти`.
